# Easy Control
Easy Control is an eCommerce website with all of eCommerce cart & payment features.

## Live Demo
(Easy Control)[https://easy-control-e9cv.vercel.app/]


### Specifications 

- Commerce.js
- Cart Features - Add, Remove, Empty and etc..
- Checkout Form
- Stripe Payment gateway
